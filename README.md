# odoo-vehicle-rental-erp
A Vehicle Rental ERP built with Odoo, featuring fleet management, booking, rental agreements, customer management, maintenance tracking, invoicing, and reporting.

## 📖 About the Project

Vehicle rental companies often rely on spreadsheets and manual processes to manage bookings, vehicles, and customer information. This project provides a centralized ERP solution to improve operational efficiency, reduce manual work, and support business growth.

The system is being developed following a professional Software Development Life Cycle (SDLC), beginning with business analysis and documentation before implementation.

## 🎯 Objectives

- Manage vehicle fleet information
- Handle customer registrations
- Process vehicle reservations and bookings
- Generate rental agreements
- Manage payments and invoices
- Track vehicle maintenance schedules
- Generate business reports and dashboards

## ✨ Planned Features
